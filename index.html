Aprendendo Flexbox

O que é Flexbox

Flexbox é um modelo de layout do CSS usado para alinhar e distribuir espaço entre os itens em um container.

Como usar?

Basta aplicar display: flex ao elemento pai e os filhos automaticamente, se organizam em linhas ou colunas.

Por que usar?

Facilita o desenvolvimento de layouts responsivos e alinhamento de elementos na tela.
